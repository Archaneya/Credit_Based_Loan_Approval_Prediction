# 💳 Credit-Based Loan Approval Prediction

A machine learning classification project that predicts whether a loan application is likely to be approved based on customer information. This project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis, feature engineering, model training, and model evaluation.

---

## 📌 Project Objective

The objective of this project is to build and compare multiple machine learning models to predict loan approval decisions using applicant details and financial information.

---

## 📂 Dataset

The dataset contains customer information such as demographic and financial details that influence loan approval.

**Target Variable:**
- Loan Status (Approved / Not Approved)
